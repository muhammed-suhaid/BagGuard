# 🛡️ BagGuard

BagGuard is a Flutter application that connects to a Bluetooth Low Energy (BLE) bag security device.

The goal of this project is to build a clean, lightweight, and reliable companion app that lets users connect to their BagGuard device, monitor its status, adjust settings, and receive alerts when movement is detected.

This project is currently under development and is being built with a production-ready architecture using Flutter and BLoC.

---

## Features

Planned features include:

- Bluetooth Low Energy (BLE) connectivity
- Scan and pair nearby devices
- Manage multiple BagGuard devices
- View connection and battery status
- Enable or disable protection mode
- Adjust movement sensitivity
- View alarm history
- Automatic device reconnection
- Modern Material 3 interface
